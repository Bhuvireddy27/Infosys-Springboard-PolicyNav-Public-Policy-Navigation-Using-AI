📌 Project Overview

Milestone 3 focuses on building an AI-powered Question & Answer Multi-Language Engine with summarization capabilities and knowledge graph integration.

The system:

🔎 Processes user queries 🌐 Retrieves relevant information 📝 Summarizes large content 🧠 Visualizes relationships using a Knowledge Graph

Knowledge graphs organize information as entities and relationships, enabling intelligent search and reasoning.

This milestone also integrates with Milestone 2 features, including authentication, UI improvements, and analytics dashboards.

⚙️ Milestone 3 Features 🌍 1. Q&A Multi-Language Engine

A question-answering engine capable of processing queries in multiple languages.

Key Functionalities

🌐 Accepts questions in multiple languages

🔄 Performs translation when required

🔎 Retrieves relevant information

💡 Generates accurate responses

📝 2. Multi-Language Summarization

The system summarizes large text into concise and readable outputs.

Features

Knowledge graphs help AI systems perform better reasoning and information retrieval.
