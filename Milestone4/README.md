🛠 Features Implemented in Milestone 4
🛡 Admin Dashboard (Management & Analytics)
The Admin Panel serves as the central command center of the system.

🔑 User Management
Admins can:

Promote users to Admin role
Lock / Unlock user accounts
Delete users from the system
📊 Activity Monitoring
Admins can monitor:

Real-time active users
Complete system activity history
📈 Data Visualization
Interactive charts provide insights into:

🤖 Most frequently used AI models
🌐 Most used languages
⭐ Most popular system features
💬 Feedback Analysis
User feedback is analyzed and visualized using a WordCloud, helping administrators quickly identify common user concerns and suggestions.

📥 Data Export
Admins can export system data including:

User feedback
History logs
User details
👤 User Dashboard & Profile Personalization
The User Dashboard improves security, customization, and user experience.

🔒 Security Features
Users can:

Update their email address
Change their account password
🧑‍🎨 Profile Personalization
Users can:

Upload a profile avatar (Display Picture)
Update their profile information
🎨 UI/UX Enhancements
Additional improvements were added to enhance the interface flow and usability, making the application more interactive and user-friendly.
